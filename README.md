# 🕊️ PIPELINE — Humanitarian Aid Game

A cooperative 3D browser game where you work as a humanitarian aid worker navigating conflict zones, building supply lines, rescuing civilians, and negotiating peace.

**Play now:** https://itsababseh.github.io/pipeline-game/

## Controls
- **WASD / Arrow Keys**: Move
- **E**: Interact (pick up supplies, rescue civilians, deliver, negotiate)
- **M**: Mute/unmute
- **Mobile**: Virtual joystick + ACT button

## Built with
- Three.js for 3D rendering
- Web Audio API for procedural sound (zero audio files)
- Pure vanilla JS (no frameworks)
